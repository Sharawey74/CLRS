JavaScript Variables Explained

Learn how variables work in JavaScript with interactive examples and clear explanations

What are Variables?

Variables are named containers that store data values. They act as references to memory locations where values are stored.

// Variable declaration and assignment
var message = "Hello, World!";
var count = 42;
var isActive = true;

Variable Naming Rules

  • Case-sensitive: myVar and myvar are different variables
  • Cannot start with a number: 1var is invalid (but var1 is valid)
  • Can contain: letters, numbers, underscores, and dollar signs
  • Cannot contain spaces: use camelCase for multi-word variables
  • Avoid reserved words: don't use JavaScript keywords like function, var, if, etc.

Valid Names

var firstName;
var total_amount;
var $price;
var _internal;
var counter1;

Invalid Names

var 1stPlace; // Starts with number
var full name; // Contains space
var function; // Reserved word
var my-var; // Hyphen not allowed

Variable Declaration

Variables must be declared before they can be used. In JavaScript, you can declare variables with var, let, or const.

Declaration with Assignment

var username = "JohnDoe";
let age = 30;
const PI = 3.14;

Declaration without Assignment

var username;
console.log(username); // undefined
