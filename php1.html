Web Development Overview

Client-Side

  • HTML - Page structure
  • CSS - Styling and layout
  • JavaScript - Interactivity
  • Runs in the user's browser
  • Manipulates the DOM

Server-Side

  • PHP, Python, Java, Ruby
  • Databases (MySQL, PostgreSQL)
  • Runs on the web server
  • Processes requests and data
  • Generates dynamic content

Client-Server Interaction

// Client (Browser) // Server // Request: GET /page.php // Process request // Query database // Generate HTML // Response: HTML page // Render page

Server-Side Languages Comparison

PHP

  • Open-source
  • Cross-platform
  • Great for websites with lower traffic
  • Simple to learn and deploy
  • Large community support

Python

  • General-purpose, high-level
  • Large standard library
  • Excellent for data-heavy applications
  • Handles large traffic volumes
  • Popular frameworks: Django, Flask

Java

  • Large-scale enterprise applications
  • Handles high volume of traffic
  • Excellent performance benchmarks
  • Strong typing and object-oriented
  • Popular framework: Spring

Ruby

  • Elegant and expressive syntax
  • Developer-friendly
  • Mature framework: Ruby on Rails
  • Convention over configuration
  • Great for rapid development

PHP Basics

PHP Code Enclosure

<?php // PHP code goes here ?>

PHP code is enclosed in special tags that tell the server to process it.

Output with echo and print

<?php echo "Hello, world!"; // Outputs: Hello, world! print "Welcome!"; // Outputs: Welcome! ?>

Both echo and print are used to output data.

PHP Output Demo

"; print "Welcome to PHP!"; ?>

PHP Variables and Constants

Variables

<?php $name = "John Doe"; // String $age = 30; // Integer $height = 5.9; // Float $isStudent = true; // Boolean echo "Name: $name, Age: $age"; ?>

Variables start with $ and are case-sensitive.

Constants

<?php define("SITE_NAME", "My PHP Site"); define("MAX_USERS", 100); echo SITE_NAME; // Outputs: My PHP Site ?>

Constants are defined with define() and are immutable.

String Concatenation

<?php $firstName = "Jane"; $lastName = "Smith"; $fullName = $firstName . " " . $lastName; echo $fullName; // Outputs: Jane Smith ?>

Conditional Statements

if Statement

<?php $age = 20; if ($age >= 18) { echo "You are an adult."; } ?>

if...else Statement

<?php $score = 75; if ($score >= 70) { echo "Pass!"; } else { echo "Fail!"; } ?>

switch Statement

<?php $day = "Monday"; switch ($day) { case "Monday": echo "Start of the week"; break; case "Friday": echo "Almost weekend!"; break; default: echo "Regular day"; } ?>
