PHP Arrays

Arrays are special variables that can hold multiple values under a single name.

Indexed Arrays

Looping Through Arrays

Indexed Array with for loop

$colors = ["Red", "Green", "Blue"]; for ($i = 0; $i < count($colors); $i++) { echo $colors[$i] . "\n"; }

Associative Array with foreach

$user = ["username" => "ahmed", "email" => "ahmed@example.com"]; foreach ($user as $key => $value) { echo "$key: $value\n"; }

Common Array Functions

count()

$arr = [1, 2, 3]; echo count($arr); // Output: 3

array_push()

$arr = [1, 2]; array_push($arr, 3); print_r($arr); // [1, 2, 3]

in_array()

$arr = ["apple", "banana"]; echo in_array("apple", $arr); // true

sort()

$arr = [3, 1, 2]; sort($arr); print_r($arr); // [1, 2, 3]

PHP Forms

GET Method

<form action="process.php" method="GET"> <input type="text" name="name"> <input type="submit"> </form> // process.php $name = $_GET['name'];

Data is visible in URL. Not secure for sensitive data.

POST Method

<form action="process.php" method="POST"> <input type="text" name="name"> <input type="submit"> </form> // process.php $name = $_POST['name'];

Data is sent in HTTP body. More secure.

Form Validation

Mandatory Fields

if (empty($_POST['name'])) { echo "Name is required"; }

Email Validation

$email = $_POST['email']; if (!filter_var($email, FILTER_VALIDATE_EMAIL)) { echo "Invalid email format"; }

PHP with MySQL

Connecting to MySQL

$servername = "localhost"; $username = "root"; $password = ""; $dbname = "myDB"; $conn = mysqli_connect($servername, $username, $password, $dbname); if (!$conn) { die("Connection failed: " . mysqli_connect_error()); }

Data Retrieval

$sql = "SELECT * FROM Customers"; $result = mysqli_query($conn, $sql); if (mysqli_num_rows($result) > 0) { while($row = mysqli_fetch_assoc($result)) { echo "ID: {$row['id']}, Name: {$row['name']}\n"; } } else { echo "No results"; } mysqli_close($conn);

PHP Sessions

Starting a Session

session_start();

Must be called before any output is sent to the browser.

Setting Session Variables

$_SESSION['username'] = "john_doe"; $_SESSION['email'] = "john@example.com";

Accessing Session Variables

echo "Hello " . $_SESSION['username']; echo "Email: " . $_SESSION['email'];

Destroying a Session

session_unset(); session_destroy();
